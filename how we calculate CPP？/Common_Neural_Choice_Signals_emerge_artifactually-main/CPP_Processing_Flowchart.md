## 三、CPP (Centroparietal Positivity) 处理详解

### CPP的定义与提取

```
┌─────────────────────────────────────────────────────────────────┐
│  CPP - Centroparietal Positivity (中央顶叶正波)                 │
└─────────────────────────────────────────────────────────────────┘

关键特征:
┌─────────────────────────────────────────────────────────────────┐
│ 1. 时间窗口: 反应前700ms到200ms (-700 to -200 ms)              │
│ 2. 锁定方式: 反应锁定 (Response-locked)                        │
│ 3. 电极位置: 中央顶叶区域 (Pz, CPz, POz等后部电极)             │
│ 4. 数据类型: 反应锁定数据 (DATR2)                              │
└─────────────────────────────────────────────────────────────────┘

提取方法 (export.m):
┌─────────────────────────────────────────────────────────────────┐
│ % CPP提取 - Pisauro方法                                         │
│ % 时间窗口: -700ms 到 -200ms (相对于反应)                      │
│ mint = -700;  % 起始时间                                       │
│ maxt = -200;  % 结束时间                                       │
│ ERPNAME = 'CPPPisauro';                                        │
│ bl = 2000;    % 基线(用于时间点计算)                           │
│                                                                 │
│ % 计算时间窗口内的平均值                                       │
│ EXPERP = nanmean(BFS(:,(mint+bl)*srate/1000:(maxt+bl)*srate/1000,:), 2);│
│                                                                 │
│ % 重塑为 [trials × electrodes] 矩阵                            │
│ EXPERP = reshape(EXPERP, [size(BFS,1), size(BFS,3)]);          │
│ EXPERP = EXPERP';                                               │
│                                                                 │
│ % 保存为 CPPPisauro700200.mat                                  │
└─────────────────────────────────────────────────────────────────┘
```

### CPP在分析中的应用

```
┌─────────────────────────────────────────────────────────────────┐
│  CPP与行为变量的关联分析                                        │
└─────────────────────────────────────────────────────────────────┘

在R脚本中 (CommonNeuralChoiceSignals_Study1.R):
┌─────────────────────────────────────────────────────────────────┐
│ # 分析CPP与RT和Value Difference的关系                          │
│ # 时间窗口: -700 到 -200 ms (反应锁定)                         │
│ # 电极: Pz (后部电极)                                          │
│                                                                 │

│                                                                 │

│ # 检验CPP是否真的反映决策变量,还是受RT伪迹影响                 │

```

### CPP处理的关键步骤

```
步骤1: 数据准备
┌─────────────────────────────────────────────────────────────────┐
│ - 使用反应锁定数据 (DATR2)                                      │
│ - 确保反应前的数据有效 (非NaN)                                  │

          │
          ▼

│ - 避免反应相关的运动伪迹 (反应后)                               │
│ - 避免早期感知成分 (反应前更早)                                 │
│ - 捕捉决策累积信号                                              │
└─────────────────────────────────────────────────────────────────┘
          │
          ▼
步骤3: 电极选择
┌─────────────────────────────────────────────────────────────────┐
│ - 中央顶叶电极: CP1, CPz, CP2, P1, Pz, P2, PO3, POz, PO4      │

┌─────────────────────────────────────────────────────────────────┐

          │
          ▼
步骤5: 统计分析
┌─────────────────────────────────────────────────────────────────┐
```

## 四、关键发现: CPP的伪迹问题

```
┌─────────────────────────────────────────────────────────────────┐
│  论文核心发现: CPP信号可能源于伪迹                              │
└─────────────────────────────────────────────────────────────────┘

问题来源:
┌─────────────────────────────────────────────────────────────────┐
│ 1. RT相关的伪迹                                                 │
│    - 反应锁定数据中,不同RT试次的时间窗口不同                   │
│    - 短RT试次: 更少的时间让决策信号累积                        │
│    - 长RT试次: 更多的时间让决策信号累积                        │
│    - 这会导致CPP与RT相关,而非真正的决策变量                    │
│                                                                 │
│ 2. 刺激锁定成分的混叠                                           │
│    - 反应锁定数据中仍包含刺激锁定的ERP成分                      │
│    - 这些成分可能与决策变量相关                                 │
│    - 传统分析方法无法分离这些成分                               │
│                                                                 │
│ 3. 时间锁定问题                                                 │
│    - 反应锁定分析假设所有试次在反应时达到相同的决策状态        │
│    - 但不同RT试次的决策过程可能不同                            │
└─────────────────────────────────────────────────────────────────┘

解决方案:
┌─────────────────────────────────────────────────────────────────┐
│ 1. Unfold解卷积                                                 │
│    - 分离刺激锁定和反应锁定的ERP成分                            │
│    - 控制RT的影响                                               │
│    - 获得纯净的决策信号                                         │
│                                                                 │
│ 2. RT中位数分组                                                 │
│    - 将试次按RT中位数分为快/慢两组                             │
│    - 比较两组的CPP信号                                          │
│    - 检验CPP是否真的反映决策变量                                │
│                                                                 │
│ 3. 控制分析                                                     │
│    - 在回归模型中控制RT                                         │
│    - 检验在控制RT后, CPP与决策变量的关系是否仍然显著           │
│                                                                 │
│ 4. 时间窗口验证                                                 │
│    - 测试不同的时间窗口                                         │
│    - 检验结果的稳健性                                           │
└─────────────────────────────────────────────────────────────────┘
```

## 六、CPP提取的具体代码实现

```matlab
% export.m中的CPP提取代码

% 设置参数
mint = -700;           % 起始时间: 反应前700ms
maxt = -200;           % 结束时间: 反应前200ms
ERPNAME = 'CPPPisauro'; % 命名
bl = 2000;             % 基线偏移 (用于时间点计算)
srate = 500;           % 采样率 500Hz

% 使用反应锁定数据
BFS = DATR2;           % DATR2是反应锁定数据矩阵

% 计算时间窗口内的平均幅度
% (mint+bl)*srate/1000 将ms转换为采样点
% nanmean忽略NaN值(伪迹试次)
EXPERP = nanmean(BFS(:,(mint+bl)*srate/1000:(maxt+bl)*srate/1000,:), 2);

% 重塑数据
EXPERP = reshape(EXPERP, [size(BFS,1), size(BFS,3)]);
EXPERP = EXPERP';      % 转置为 [trials × electrodes]

% 保存结果
save(sprintf('%s%s%s%d%d.mat', PATH, EXPORTPATH, ERPNAME, abs(mint), abs(maxt)), ...
     sprintf('%s%d%d', ERPNAME, abs(mint), abs(maxt)), '-v6');
```

## 七、重要说明

1. **数据维度**:
   - BFS/DATR2: [61 electrodes × timepoints × trials]
   - 61个电极包括60个EEG电极 + 1个Cz
   - 时间点数量取决于采样率和时间窗口

2. **时间窗口选择的原因**:
   - -700到-200ms避免了反应相关的运动伪迹
   - 这个窗口被认为包含决策累积信号
   - 与Pisauro et al. (2017)的方法一致

3. **电极选择**:
   - 中央顶叶电极 (Pz及其周围)
   - 这是CPP的典型分布位置
   - 可以选择单个电极或多个电极平均

4. **关键发现**:
   - 论文发现CPP信号可能主要反映RT相关的伪迹
   - 在控制RT后, CPP与决策变量的关系减弱或消失
   - 这挑战了CPP作为决策累积信号的观点
